# 控制台总览

本章节介绍 tokenfor.me Web 控制台的整体结构与常用入口。

> 说明：菜单名称以线上实际界面为准。本文使用的名称与结构仅为示例，后续会根据线上中/英文菜单名称保持一致。

## 顶部导航与侧边栏

登录 `https://tokenfor.me` 后，你通常会看到：

- 顶部导航栏：账号信息、语言切换、通知、退出登录；
- 左侧侧边栏（当前菜单示例）：
  - 仪表盘（Dashboard）
  - API 密钥（API Keys）
  - 使用记录（Usage）
  - 我的订阅（My Subscriptions）
  - 兑换（Redeem）
  - 个人资料（Profile）

> 本帮助文档目前重点介绍「仪表盘 / API 密钥 / 使用记录」相关内容，
> 「我的订阅 / 兑换」如需说明，可在未来单独扩展章节。
## 仪表盘 / Dashboard

- 近期请求趋势、常用分组、费用估算、快捷操作入口等。

## API 管理 / Keys

- 创建 Key、查看供应商/分组、启用/禁用/删除、备注等。

## 使用情况 / Usage

- 时间维度统计、按 Key/分组/供应商拆分；
- 支持导出 CSV 用量明细（如界面提供相应按钮）。

## 账单 / Billing

- 当前账期与历史账单、支付记录、余额/额度（如有）。

## 倍率说明 / Multipliers

- 解释倍率（折扣系数）含义与示例；
- 不同分组的倍率与体验说明；
- 倍率申请与调整流程（联系销售/工作人员）。

## 货币与结算 / Currency & Settlement

- 以美元 $ 为展示/结算单位；
- 其他货币在充值时按实时汇率折算；
- 对账单请在「使用情况」导出 CSV；
- 输入/输出/缓存 token 数据由模型公司接口返回；
- 结算价格与官方保持一致（以官方为准）。

## 账号与安全 / Account & Security

- 密码、安全与登录记录、双重认证（如有）。

## 帮助与支持 / Support

- 文档、客服与售后、公告与变更通知。
