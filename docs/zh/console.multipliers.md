# 倍率说明（Multipliers）

## 倍率是什么？

- 倍率是相对于标准价格的折扣系数；
- 示例：九折 = 0.9；
- 不同的分组（渠道）对应不同的倍率与体验；
- 倍率由销售或工作人员在开通/调整时为你配置。

## 为什么会有不同分组？

- 同一家供应商（如 OpenAI / Anthropic / Gemini）可能接入多条通道；
- 不同通道在网络、稳定性、时延与可用性上存在差异；
- 为了透明计费与体验选择，引入「分组 + 倍率」的管理方式。

## 如何查看我当前的倍率？

- 在控制台中查看 Key 所属的供应商与分组；
- 分组对应的倍率由系统或工作人员标注（以页面显示或合同约定为准）。

## 如何申请/调整倍率？

- 请联系销售或工作人员，沟通你的业务场景与期望体验；
- 按需评估后为你配置合适的分组与倍率。
