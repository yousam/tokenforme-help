# 控制台总览

本章节介绍 tokenfor.me Web 控制台的整体结构，帮助你快速找到创建 API、查看用量和账单、进行账号管理等常用入口。

> 说明：以下菜单名称为示例，实际名称请以线上界面为准，但整体功能区划分通常类似。

## 顶部导航与侧边栏

登录 `https://tokenfor.me` 控制台后，你通常会看到：

- 顶部导航栏：展示当前账号信息、语言切换、通知和退出登录等入口；
- 左侧侧边栏：按照功能模块划分，例如：
  - 仪表盘 / Dashboard
  - API 管理 / Keys
  - 使用情况 / Usage
  - 账单 / Billing
  - 账号与安全 / Account & Security
  - 帮助与支持 / Support

## 仪表盘 / Dashboard

仪表盘通常用于展示账号下的整体使用概况，例如：

- 近期请求数量与趋势；
- 最近使用的模型或通道；
- 费用估算或当前账期消费情况；
- 常用功能的快捷入口（创建 API Key、查看账单、联系支持等）。

这部分可以帮助你快速了解当前账号是否有异常用量、是否接近配额等。

## API 管理 / Keys

这是普通用户最常用的模块之一，用于：

- 创建新的 API Key；
- 查看现有 Key 列表（名称、创建时间、状态、用途备注等）；
- 启用 / 禁用 / 删除某个 Key；
- 为 Key 配置可用的模型路由（例如 OpenAI、Anthropic、Gemini 等）。

具体操作步骤会在「创建与管理 API」章节详细展开。

## 使用情况 / Usage

在「使用情况」页面，你可以：

- 查看一段时间内的请求总量、耗费的 tokens 或其他计费指标；
- 按模型、按 API Key、按时间维度（天 / 周 / 月）进行筛选和统计；
- 导出部分统计数据（如果支持）。

这对于个人用户和企业用户的成本控制都非常重要。

## 账单 / Billing

「账单」模块主要用于：

- 查看当前计费周期内的消费详情；
- 查看历史账单和支付记录；
- 了解当前余额 / 信用额度（如果有）。

关于充值方式和联系方式，请参考「充值与付费说明」章节。

## 账号与安全 / Account & Security

在该模块中，你可以：

- 修改登录密码、绑定或更换邮箱 / 手机；
- 开启或关闭双重认证（如果支持）；
- 查看最近账号登录记录和安全提醒。

建议定期检查安全设置，保障账号安全。

## 帮助与支持 / Support

通常会提供：

- 文档入口链接（你现在正在阅读的就是其中之一）；
- 联系客服或提交工单的入口；
- 常见问题链接、公告与变更通知。

如果在使用过程中遇到问题，可以优先查阅文档和 FAQ，如仍无法解决，再通过售后支持渠道联系人工客服。
