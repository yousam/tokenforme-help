# 常见问题 FAQ

本章节汇总了个人和企业用户在使用 tokenfor.me 过程中的常见问题及简要解答。

## 账号与登录相关

### 忘记密码怎么办？

在登录页面点击「忘记密码」，按照提示通过邮箱或其他方式重置密码。如果仍有问题，可通过文末的售后支持渠道联系人工协助。

### 可以一个账号多人共用吗？

不推荐。建议：

- 个人用户为自己单独注册账号；
- 团队 / 企业用户为每位成员创建独立账号，并通过组织 / 团队功能统一管理权限和用量。

## API 使用相关

### 已经在工具中配置了 API 地址（Base URL）和 API Key，但请求失败怎么办？

请依次检查：

1. API 地址（Base URL）是否按照「API 密钥」章节中「API 地址」小节的说明配置（为所在区域选择正确的调用域名）；
2. API Key 是否正确粘贴（注意不要多空格或缺字符）；
3. 该 Key 是否已在控制台中启用，且未被删除；
4. 当前网络环境是否可以正常访问外网（包括你在「API 密钥」章节中「API 地址」小节所配置的调用域名）。

如仍无法解决，请记录错误信息和时间，并联系售后支持。

### 一个 Key 可以同时使用 OpenAI、Anthropic、Gemini 等多家模型吗？

不可以。**一个 Key 只对应一个供应商（AI 公司）下的一个分组**。如果需要同时使用多家模型，请分别创建不同的 Key，并在各自的工具里配置相应的 Key。
### 模型名称如何填写？

- 一般情况下，可以直接使用你所熟悉的模型名称（如 `gpt-4`, `claude-3-opus`, `gemini-1.5-pro` 等）；
- 若出现「模型不存在」或类似错误，请检查：
  - 该模型是否在 tokenfor.me 控制台中已启用；
  - 是否使用了正确的模型标识符（以控制台或官方文档为准）。

## 账单与费用相关

### 费用是如何计算的？

费用通常由多家模型供应商的计费规则叠加而来，tokenfor.me 会在账单中给出对应用量和估算费用。具体计费规则可参考控制台中的说明和官方公告。

### 如何进行充值？

- 个人客户：
  - 请通过「售后与客服支持」章节中的「联系方式」小节联系客服（客服QQ / 客服Wechat / 大客户Email）。
  - 了解最新优惠信息和在线充值方式；
- 集团 / 企业客户：
  - 请邮件联系：`bill@tokenfor.me`；
  - 获取适配企业的充值、结算与合同方案。

### 有折扣或优惠活动吗？

有。具体活动内容会不定期调整，请参考「充值与优惠」或通过上述联系方式咨询。

## 安全与权限相关

### API Key 泄露了怎么办？

请立即：

1. 登录 `https://tokenfor.me` 控制台；
2. 禁用或删除疑似泄露的 Key；
3. 创建新的 Key，并在各个工具中更新配置；
4. 如怀疑存在异常用量，请在「使用情况 / Usage」中查看明细，并联系售后支持。

### 是否支持多级权限和团队管理？

如你有多级权限或团队管理相关需求（如不同部门、子项目的分权管理），建议直接联系销售或售后支持，获取最新的企业功能说明和开通方式。

## 其他问题

如本页未覆盖你遇到的问题，建议：

1. 先查阅「故障排查」章节；
2. 若仍无法解决，按照「售后支持」章节提供的方式联系人工客服。
