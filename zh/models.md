# 模型及计费

本章节汇总当前主流 AI 厂商开放的模型列表、能力说明及计费规则，方便统一查询与对接。

## 模型说明

- 系统默认接入各 AI 厂商的**最新可用模型**
- 不同 API Key 绑定不同厂商分组
- **模型可用性取决于分组授权情况**
- 所有模型调用均保持与官方 API 高度兼容

## 计费规则说明

所有模型的计费标准遵循对应厂商官方定价，平台不做额外修改。

### Token 计费维度

通常分为三类：

| 类型 | 说明 |
|------|------|
| Input Tokens | 用户输入消耗 |
| Output Tokens | 模型生成消耗 |
| Cache Tokens | 上下文缓存（部分模型支持） |

### 计费原则

- **按 Token 使用量计费**
- **不同模型单价不同**
- **输入 / 输出价格通常不同**
- **长上下文模型可能启用 Cache 计费优化**

### 官方对齐说明

- ✅ 与各厂商官方价格保持一致   
- ✅ 支持最新模型价格自动同步   

## 注意事项

- 不同 Key 所属分组，**可用模型可能不同**
- 某些模型可能需要**单独开通权限**
- 长上下文模型（如 Gemini / Claude）可能涉及额外费用
- 推理模型（如 o 系列）通常价格更高

## 后续更新

- 模型清单将持续更新
- 新模型上线后将第一时间同步
- 计费变动以官方公告为准
