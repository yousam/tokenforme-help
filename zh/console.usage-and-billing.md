# 使用情况与账单

本章节介绍如何在 tokenfor.me 控制台中查看请求用量、费用估算，以及如何进行充值和了解优惠信息。

## 使用情况 / Usage

在「使用情况 / Usage」页面，你可以从多个维度了解账号的整体使用情况：

- 时间维度：按天、周、月查看调用趋势；
- 模型维度：不同模型（如 OpenAI、Anthropic、Gemini 等）的用量分布；
- Key 维度：不同 API Key 的调用次数、消耗量和费用占比。

这些信息可以帮助你：

- 判断是否存在异常调用（例如短时间内调用激增）；
- 分析不同项目或成员的使用情况；
- 为后续成本优化提供依据。

## 账单 / Billing

在「账单 / Billing」页面，你通常可以：

- 查看当前计费周期：
  - 本周期内的总消费情况；
  - 已支付金额与未结清金额（如有）；
- 查看历史账单：
  - 每个账单周期的用量与对应费用；
  - 支付记录与发票信息（如支持）。

如需进一步财务或报销用途，可导出账单明细或联系支持获取更详细的账单数据。

- WeChat：`zzbar2025`
- QQ：`3928789892`

你可以通过上述联系方式：

- 获取在线充值的具体操作指引；
- 咨询余额、账期等相关问题。

### 集团 / 企业客户充值

对于集团或企业客户，建议采用邮件沟通方式，以便获得更完整的方案与对接流程：

- 邮件联系方式：`bill@tokenfor.me`

通过邮件，你可以：

- 了解企业级充值与结算方式；
- 协商定制套餐与专属折扣；
- 获取适用于财务、采购流程的正式材料。

## 成本控制建议

- 按 Key 区分环境：为「开发 / 测试 / 生产」分别创建不同 Key，便于统计和控制；
- 配合用量统计：定期检查 Usage 页面，关注异常峰值与某些模型的高成本占比；
- 与团队约定使用规范：例如限制某些高成本模型的使用频率。
