# 货币与结算

## 结算币种

- 系统以 **美元 ($)** 作为展示与结算单位；
- 其他货币在充值时按实时汇率折算为美元；
- 账单与费用请以控制台展示与导出数据为准。

## 用量与账单数据来源

- 输入/输出/缓存 token 数据均由大模型公司接口返回；
- 各模型 token 价格与官方保持一致（以官方为准）。

## 导出对账数据

- 请在「使用情况 / Usage」页面使用 **导出 CSV** 功能，获取按时间/Key/分组拆分的明细；
- 建议结合账单周期进行归档保存，便于财务对账与审计。

## 充值与发票

- 个人客户充值与优惠：
如需了解更多结算方式、合同及发票等信息，请通过「售后与客服支持」章节中的「联系方式」小节联系客服（客服QQ / 客服Email）。
- 集团/企业客户：
  - 邮件：参考「概览-联系方式」章节中的「客服Email」（可沟通结算、合同、发票等事宜）
