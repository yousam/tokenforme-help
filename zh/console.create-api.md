# 创建与管理 API

本章节详细介绍如何在 tokenfor.me 控制台中创建、查看和管理 API Key，以及如何为 Key 选择供应商与分组（通道）。

## 创建新的 API Key（绑定 AI 厂商分组）

1. 登录 `https://tokenfor.me` 控制台。
2. 在左侧菜单进入「API 密钥」。
3. 点击「创建 API Key」。
4. 填写以下信息（以实际界面为准）：
   - 名称：如 `personal-dev`、`backend-service`；
   - 分组（倍率）：如「GPT Standard」等（每个分组隶属于某个 AI 厂商）；
5. 确认保存后，系统会生成一个新的 API Key，请妥善保管。
> 重要规则：
> - **一个 Key 只对应一个 AI 厂商的一个分组**；
> - 如果你需要同时使用多家 AI 厂商或同一 AI 厂商下的不同分组，请「分别创建新的 Key」。

> 模型说明：目前会为你开放各 AI 厂商最新的模型，
> 具体模型列表和说明请参考「模型」板块（后续会在文档中单独补充）。

## 管理现有 Key

- 查看 Key 列表：名称、供应商、分组、创建时间、状态、最近使用；
- 编辑 Key：修改名称/备注、调整分组（如允许）；
- 启用/禁用/删除 Key：泄露或不再使用时建议禁用或删除；
- 分组与倍率（如显示）：可查看当前分组对应的倍率信息（仅供参考，以销售/开通为准）。

## 使用建议

- 按用途分 Key：
  - 个人开发、项目 A、项目 B、生产环境等分别使用不同 Key；
- 按供应商/分组分 Key：
  - 例如 OpenAI（GPT Standard）与 Anthropic（某分组）分别创建 Key；
- 定期审查：
  - 清理长期不用或疑似泄露的 Key，避免不必要成本与风险。
